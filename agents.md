# Multi-Agent Development Roles

## Leader Agent

Coordinates tasks and milestones. Assigns work to other agents, tracks progress, resolves blockers, and ensures deliverables align with project goals and timeline.

## UX Designer Agent

Defines player flow, accessibility, and UI layout. Specifies screen hierarchy, touch targets, visual feedback, and one-handed portrait usability for mobile play.

## Developer Agent

Implements Phaser 3 code, manages physics and obstacle spawning. Owns game scenes, input handling, collision detection, scoring logic, and build output for WebView deployment.

## QA Agent

Tests gameplay balance, bug reports, mobile responsiveness. Validates jump feel, difficulty curve, performance on target devices, and cross-platform behavior in WebView.

## Content Agent

Manages assets (sprites, audio, backgrounds). Sources or creates dog animations, obstacles, parallax layers, UI elements, and sound effects optimized for fast mobile loading.
